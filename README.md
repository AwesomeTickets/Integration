# Integration

Integration and deployment scripts of AwesomeTickets system.

## Installation

- Run integration test

    ```bash
    $ ./integration_test.sh
    ```

- Deploy

    ```bash
    $ ./deploy.sh
    ```

- Enter a container

    ```bash
    $ ./enter_container.sh
    ```

**Notes:** Use param `-h` to see the usages of these scripts.
